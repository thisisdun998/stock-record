<!-- Plugin description -->
Stock Record is a plugin for IntelliJ IDEA that helps you monitor stock market quotes in real-time.
这是一个用于 IntelliJ IDEA 的股票行情监视插件。
**功能特点：**
1. 📊 **实时行情**：支持沪深 A 股、港股实时行情查看。
2. 📝 **自选管理**：支持添加、删除自选股。
3. 🔄 **实时显示**：实时显示涨跌幅和价格。
4. 📈 **指数监控**：支持大盘指数展示。
